Formatando o Texto com Estilos

font-family

O que faz: escolhe a família da fonte do texto.

Código:

font-family: Arial;

Resultado:

Exemplo usando a fonte Arial

font-style

O que faz: altera o estilo da fonte.

font-style: italic;

Resultado:

Texto em itálico

font-weight

O que faz: define a espessura do texto.

font-weight: bold;

Resultado:

Texto em negrito

font-size

O que faz: altera o tamanho da fonte.

font-size: 12px;

Resultado:

Texto com tamanho 12px

line-height

O que faz: controla a altura entre as linhas.

line-height: 1.6;

Resultado:

Exemplo com múltiplas linhas.
Observe o espaçamento entre elas.
Isso melhora a leitura.

color

O que faz: altera a cor do texto.

color: red;

Resultado:

Texto vermelho

background

O que faz: altera a cor de fundo.

background: blue;

Resultado:

Texto com fundo azul

text-align

O que faz: define o alinhamento do texto dentro do elemento.

Código:

text-align: left;

Resultado:

Este texto está alinhado à esquerda.

Código:

text-align: center;

Resultado:

Este texto está centralizado.

Código:

text-align: right;

Resultado:

Este texto está alinhado à direita.

Código:

text-align: justify;

Resultado:

Este texto está justificado, ocupando todo o espaço disponível da linha, alinhando as margens esquerda e direita.

text-transform

A propriedade text-transform altera a forma como o texto é exibido.

Código:

text-transform: capitalize;

Resultado:

capitalize → cada palavra começa com letra maiúscula

Código:

text-transform: uppercase;

Resultado:

uppercase → todo o texto fica em letras maiúsculas

Código:

text-transform: lowercase;

Resultado:

LOWERCASE → todo o texto fica em letras minúsculas
