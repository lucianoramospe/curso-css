CSS Grid

O que é CSS Grid?

O CSS Grid é um sistema de layout usado para organizar elementos em linhas e colunas.

Ele é muito utilizado para criar estruturas profissionais de páginas, como cards, galerias e dashboards.

Ativando o Grid

Para transformar um container em grade, usamos: 


display: grid;

Definindo Colunas

Usamos grid-template-columns para definir a quantidade de colunas. 


grid-template-columns: 1fr 1fr 1fr;

Resultado:

1
2
3
4
5
6

Espaçamento com gap

O gap cria espaço entre as caixas. 


gap: 20px;

Explicando o 1fr

A unidade fr significa “fração do espaço disponível”.

Exemplo: 


grid-template-columns: 1fr 1fr 1fr;

Significa 3 colunas com o mesmo tamanho.
