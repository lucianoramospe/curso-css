Modelo de Caixa (Margem, Preenchimento e Borda)

O que é Modelo de Caixa (Box Model)?

Todo elemento HTML é representado como uma caixa. Essa caixa é composta por:

  • Content → conteúdo interno
  • Padding → espaço interno
  • Border → borda
  • Margin → espaço externo

Exemplo Visual

Código:


div {
    margin: 20px;
    padding: 30px;
    border: 3px solid blue;
}

Resultado:

Conteúdo da caixa

Margin (Margem)

A propriedade margin define o espaço externo do elemento. 

margin: 20px;

Cria espaço ao redor da caixa.

Padding (Preenchimento)

A propriedade padding define o espaço interno. 

padding: 30px;

Cria espaço entre o conteúdo e a borda.

Border (Bordas)

A propriedade border define a borda do elemento. 

border: 3px solid blue;

Cria uma borda azul ao redor da caixa.
