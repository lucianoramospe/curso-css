Cores no CSS

As cores são utilizadas para alterar a aparência do texto, do fundo, das bordas e de outros elementos da página.

Usando nomes de cores

O que faz: utiliza nomes prontos em inglês.

color: red;

Resultado:

Este texto está vermelho

Cor de fundo

O que faz: altera a cor do fundo do elemento.

background-color: blue;

Resultado:

Texto com fundo azul

Cores em RGB

O que faz: define cores usando vermelho, verde e azul.

color: rgb(255, 0, 0);

Resultado:

Texto usando RGB

Cores Hexadecimais

O que faz: usa código hexadecimal para representar cores.

color: #2563eb;

Resultado:

Texto com cor hexadecimal
