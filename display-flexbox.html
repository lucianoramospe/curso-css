Display e Flexbox

O que é a propriedade display?

A propriedade display define como um elemento HTML será exibido na página.

display: block

O que faz: ocupa toda a largura disponível da linha.

display: block;

Resultado:

Elemento Block

display: inline

O que faz: ocupa apenas o espaço do conteúdo.

display: inline;

Resultado:

Inline 1 Inline 2

display: flex

O Flexbox é usado para organizar elementos em linha e criar layouts modernos. 


display: flex;
justify-content: center;
gap: 20px;

Resultado:

1
2
3

justify-content

Controla o alinhamento horizontal dos elementos. 

justify-content: center;

Centraliza os itens horizontalmente.

align-items

Controla o alinhamento vertical dos elementos. 

align-items: center;

Centraliza os itens verticalmente.

gap

Cria espaço entre os elementos. 

gap: 20px;

Espaçamento entre as caixas.
