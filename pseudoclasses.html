Pseudo-classes (:hover e :focus)

O que são Pseudo-classes?

As pseudo-classes permitem aplicar estilos em situações específicas, como quando o mouse passa sobre um elemento ou quando um campo recebe foco.

:hover

O que faz: aplica estilo quando o mouse passa por cima.

Código:


.botao:hover {
    background-color: #1d4ed8;
    transform: scale(1.05);
}

Resultado:

:focus

O que faz: aplica estilo quando o campo é selecionado.

Código:


input:focus {
    border: 3px solid #2563eb;
}

Resultado:

Resumo

  • :hover → mouse sobre o elemento
  • :focus → campo selecionado
