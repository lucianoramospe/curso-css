# Curso de CSS

Curso completo de CSS para iniciantes.

##  Conteúdo

- Introdução ao CSS
- Formas de aplicar CSS
- Seletores
- Formatação de texto
- Box Model
- Cores e fundos
- Flexbox
- Pseudo-classes
- Responsividade
- CSS Grid
- Projeto final

## Como usar

1. Baixe o projeto
2. Abra o arquivo `index.html`
3. Navegue pelas aulas

## Público

Estudantes do Ensino Médio Técnico em Desenvolvimento de Sistemas.
