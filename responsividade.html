Responsividade e @media

O que é Responsividade?

Responsividade é a capacidade do site se adaptar a diferentes tamanhos de tela, como computador, tablet e celular.

O que é @media?

A regra @media é usada para aplicar estilos diferentes de acordo com o tamanho da tela do dispositivo.

Isso é muito importante para que a página fique bonita em:

  • Computadores
  • Tablets
  • Celulares

Em outras palavras, o CSS muda automaticamente quando a tela diminui ou aumenta.

Código:


@media (max-width: 768px) {
    body {
        font-size: 18px;
    }
}

Explicando o código:

@media → inicia a regra de responsividade

max-width: 768px → significa que esse estilo será aplicado quando a largura da tela for de até 768 pixels.

font-size: 18px → altera o tamanho da fonte para 18 pixels.
